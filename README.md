**Driftn Goose:**
This project was made locally and does not have version control or commits in the github. All the files, demo video, and markup report will be accessible from here:

**Description:**
This is a 2D lane runner game made with the theme of vaporwave where a goose flys through a vaporwave styled space world and switches lanes to avoid getting hit asteroids.
This helps with my career because it helps further my knowledge of backend design so I can create more backend friendly UX/UI designs.

**Play instructions:**
https://drive.google.com/drive/folders/1EevyThTVNTvN3x1eVJOSK1vDLfOFsYx9?usp=sharing
Visit the drive link and download the game files. Use the path: My project/Build/My project.exe and run the executable. Then the game will open and the round will automatically start
Controls:
Move up - w or up arrow
Move down - s or down arrow
